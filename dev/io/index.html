Mesh I/O

Loading and saving of files can be done through the FileIO functions load and save.

File loading

Files can be loaded in this way:

using FileIO
bust = load("beethoven.ply")

The following formats should be supported: .stl (ASCII and binary) .ply (ASCII and little-endian/big-endian binary).

File saving

Files can be saved in this way:

using FileIO
save("foo.stl", object)

The following formats are supported: .stl (ASCII) and .ply (ASCII) for volumes; .svg for shapes.

Image file formats are also supported through CairoMakie's plot functions.
