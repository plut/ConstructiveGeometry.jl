CSG operations

Boolean operations

Base.union — Method
union(a::AbstractGeometry{D}...)

Computes the union of several solids. The dimensions must match.

source
Base.intersect — Method
intersect(a::AbstractGeometry{D}...)

Computes the intersection of several solids. Mismatched dimensions are allowed; 3d solids will be intersected with the (z=0) plane via the slice() operation.

source
Base.setdiff — Method
setdiff(a::AbstractGeometry, b::AbstractGeometry)

Computes the difference of two solids. The following dimensions are allowed: (2,2), (3,3), and (2,3) (in the last case, the 3d object will be intersected with the (z=0) plane via the slice() operation).

setdiff([a...], [b...])

Shorthand for setdiff(union(a...), union(b...)).
